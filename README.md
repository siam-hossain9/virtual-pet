# Virtual Pet Website

Virtual Pet is a browser-based pet game built with Next.js and TypeScript. You can care for your pet, play with it, customize it, collect rewards, and track progress through a simple interactive interface.

## Features

- Home dashboard for core pet status
- Play screen for interactions
- Customize page for pet appearance changes
- Shop and profile pages
- Reusable UI components for stats, rewards, navigation, and character rendering
- Centralized state management for pet data

## Tech Stack

- Next.js (App Router)
- TypeScript
- React
- CSS

## How to Run This Website

1. Install Node.js 18.18 or later (Node.js 20 recommended).
2. Open this project folder in your terminal.
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser at:

http://localhost:3000

## Build for Production

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```

## Project Structure

- src/app: App Router pages (home, play, customize, shop, profile)
- src/components: Reusable UI components
- src/lib: Utility and domain logic
- src/store: Application state store
- public/pets: Static pet assets
